Exercise 1: Online Bookstore - Setting Up RESTful Services

Step 1: Create a Spring Boot Project:

Open Spring Initializr.
Project: Maven
Language: Java
Spring Boot Version: 3.3.2
Group: com.example
Artifact: BookstoreAPI
Name: BookstoreAPI
Package Name: com.example.bookstoreapi
Packaging: Jar
Java Version: 17

Add Dependencies:
Spring Web
Spring Boot DevTools
Lombok

Generate the Project:
Click Generate to download the project as a .zip file.

Extract the Project:
Unzip the project. 

Step 2: Import the Project into Intellij:

Launch Intellij IDE.
Browse to the location of the unzipped project and select the project.
Click Finish to import the project.
